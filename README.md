## About this project

Simple CRUD developed using Laravel 8 and Bootstrap 4.

Functionalities:
- List all registered products;
- Show a single product;
- Create products with name and datail;
- Edit products;
- Delete product.

Details:
- Check if fields have some value;
- Route controll;
- Very nice usability
